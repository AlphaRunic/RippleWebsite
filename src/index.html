Why Was Ripple Created?

Ripple was created because I felt Discord lacked a consistent all-purpose bot (that's actually all-purpose). Obviously there's no scarcity of all-purpose bots in Discord, but another flawless TypeScript bot just completes the picture.

Using Source Code

Ripple can be cloned directly from it's repository for the latest version. To make things easier, an NPM package is provided to help install everything cleanly. In this package, a "ripple" binary is also included. Install via npm i ripple-discord-ts -g or visit the NPM package.

Features

Ripple is a lightweight, yet command-heavy bot. It has 100+ commands, and growing.

Customization

Ripple has a plethora of commands to help personalize your server to the maximum. These commands include utility to set a custom welcome message, an automatic role to be applied upon a user joining, bot prefix, chat revive, level system, and more!

Quoted Arguments

Ripple supports a system called quoted arguments. This means that if you pass an argument to a command, and that argument is a string, you can surround it in quotation marks to treat space-delimited text as one argument.

Type Coercion

Ripple command arguments have a function called type coercion. The purpose of this function is to coerce an argument (which is originally a string) to a different type. For example, in the command
::urban "pog champ" 5 , the value "5" is being used to denote what page Urban Dictionary should search. When you input the number 5 into Discord, it's automatically a string (plain text), but Ripple coerces it's type to a number, because the resultNumber argument is defined as a number in the Urban command.
